About My Projects

Microcontrollers Directory: Various sketches written in the Arduino IDE for the completion of CPET.202 Microcontrollers Systems Lab.

Respiratory Rate as a Clinical Indicator: Implentation of various filters written in the Arduino IDE in conjunction with a dithering circuit and an alarm that allowed for the detection of abnormal breathing rates which would then sound an alarm. Report here.

Sitemap XML Parser: Parses the sitemap.xml of a website and updates at a specified polling interval. Stores the first pass in an array and updates a new array for each subsequent pass. Tweepy library integration allows for the posting of tweets when urls are removed or added. ( Seen on my anonymous sneaker twitter account ) Added option of json parsing for each newly added url allows for the display of variants and quantities which can be used specifically on the shopify platform in preloading carts via permalinks.

SnapSave: Integrated mitmproxy in python allows for the capture flows which can then be replicated with the extensive parsing and urllib2. Requires proper installation of certificate via visiting mitm.it on your device. Also requires mimproxy for the actual subprocess. Follow on screen instructions to setup the proxy on your device. Current version mitigates MITM via certificate pinning. Read here.

To circumvent this, on your device run the following command via MobileTerminal:
wget bit.ly/SSLKillSwitch2Deb -O SSLKillSwitch2_0.10.deb

Now that the debian package has been downloaded. Install this package as a root user via the command:
dpkg -i SSLKillSwitch2_0.10.deb

Run:
killall -HUP SpringBoard

After respring, go into Settings->SSL Kill Switch->Enable "Disable Certificate Validation"
Make sure to close SnapChat if running in the background.

Proceed with the application as normal.

IG_Last20: Via a series of page scrapes/parses, users available via my access_token, ( users viewable by @zweed4u ) have their last 20 most recent media shown with caption, filter and media url.
